# Teslo API

1. Clonar en proyecto
2. ```npm install```
3. Clonar ```.env.template``` y renobrar a ```.env```
4. Cambiar las variables de entorno
5. Levantar la base de datos
```
docker-compose up -d
```
6. Subir el modo desarrollo ```npm run start:dev```
